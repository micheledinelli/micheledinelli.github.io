### Cosa significa l'opzione `-cp`?

L'opzione `-cp` nel comando `java` sta per "classpath". Il classpath è l'elenco di directory o file JAR che Java utilizza per trovare le classi richieste durante l'esecuzione di un programma.

Nel comando `java -cp . EsercizioN arg1`, l'opzione `-cp .` imposta il classpath corrente (`.`) per includere la directory corrente, che è dove il comando si sta eseguendo. Questo significa che Java cercherà le classi necessarie all'interno della directory corrente durante l'esecuzione del programma `EsercizioN`.

La parte `EsercizioN arg1` indica il nome della classe da eseguire (`EsercizioN`) e gli eventuali argomenti da passare al programma (`arg1`).
